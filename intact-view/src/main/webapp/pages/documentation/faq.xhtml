Frequently Asked Questions (FAQ)

Direct linking into the intact web site can be achieved using one of the following methods:
1. Interaction tab using a free text query: You can open the interaction tab and display a list of interaction matching you query by using the URL below: http://www.ebi.ac.uk/intact/pages/interactions/interactions.xhtml?query=bbc1 The query should conform to the [MIQL specification]. 2. Details tab by experiment AC Given an experiment AC, one can open the details view, for instance: http://www.ebi.ac.uk/intact/pages/details/details.xhtml?experimentAc=EBI-768904 Note: this will not show any interaction details but only experiment and respective publication. 3. Details tab by interaction AC Given an interaction AC, one can open the detail view, for instance: http://www.ebi.ac.uk/intact/pages/details/details.xhtml?interactionAc=EBI-2307627 4. Interaction given a pair of interactor Given two interactors' AC, one can open the interaction detail view, for instance: http://www.ebi.ac.uk/intact/pages/details/details.xhtml?binary=EBI-3437,EBI-2750
