DrugPort provides a freely available, open source database system and analysis tools for drug-target interactions. DrugPort combines detailed drug (i.e. chemical, pharmacological, and pharmaceutical) data with molecular target (i.e. sequence, structure, and pathway) information. Version 0.9 of the database is structured around the known druggable genome, an expanding list of ~4000 known and predicted drug targets. The stated aim of DrugPort is to give the entire drug-discovery community access to all publicly available chemistry space and molecular targetspace. Using this information, researchers will be able to use computational methods to select the most efficient small molecule libraries and targets to support high throughput screening, cross screening and structure activity relationship (SAR) analysis, to support drug discovery, safety and efficacy studies, respectively.
