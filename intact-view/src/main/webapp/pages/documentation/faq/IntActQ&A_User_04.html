What is PSI-MI XML and why use it?



Proteomics Standards Initiative - Molecular Interaction


PSI-MI is an XML format supported by the Human Proteome Organisation storing and exchanging molecular interaction data. The Proteomics Standards Initiative was founded at the HUPO meeting in Washington, April 28-29, 2002 (see Science, 296, 827) and has been tasked with providing data standards in the field of Proteomics. As a first step, the PSI is concentrating on mass spectrometry and protein-protein interaction data.

The PSI-MI Format
An important feature of the PSI-MI format is that its structure enables the data to be held in multiple files which can be amalgamated, thus, allowing the exchange/download of species or experimental specific files. Below is a model of the PSI-MI format.

As can be seen the top level element "entry" enables the partitioning of the data. The children of this element shown on the right of the picture describe the details of the entry; where it comes from (source), the different experiments (experimentList), the different interactor molecules (interactorList), the interactions those molecules partake in (interactionList) and any other details that may be important for researchers using the data set to know (attributeList). For further details of the PSI-MI format please refer to this link. Below is an example of a PSI-MI XML file.

XML stands for Extensible Markup Language. The navy-blue words are the tags that mark-up the data and enable it to be assigned to its correct place in the database. (This is similar to the HyperText Markup Language (HTML) that is used to write web-pages). Blue letters which are followed by an = sign are attributes that define the type of data (e.g. “db=” in this case referring to a database). The information encapsulated in quotes, coloured green, (e.g. "interpro", "IPR006663") is the data held in the database. To view an entire dataset download a PSI-MI XML file from the IntAct data repository.
