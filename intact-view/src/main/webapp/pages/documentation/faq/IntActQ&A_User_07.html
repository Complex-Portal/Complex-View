What protein(s) interact with my chosen protein?

IntAct can be searched by:

Input Example
Gene name cdc53* or cdc53_yeast
IntAct Accession EBI-4321
UniProtKB Accession Q12018
InterPro Accession IPR001373
Gene Ontology ID GO:0000152
PubMed ID 11805837


Gene name       [back2list]
The gene name is usually an alphanumeric code which is an abbreviation relevant to the function of the gene. After the code a number may be added to differentiate between different members of the same family. The numbers assigned to the gene normally have no functional meaning but simply represent the order in which the genes were discovered.


IntAct Accession Number       [back2list]
To co-ordinate tables in the IntAct database all protein entries are given a unique IntAct Accession Number. Entering this into the IntAct search engine will take you to a list of interaction partners for that protein.


UniProtKB Accession Number       [back2list]
The UniProt KnowledgeBase is a comprehensive, annotated protein sequence knowledgebase with extensive cross-references ( Bairoch et al. 2005 ). Two important sections of UniProtKB are UniProt/Swiss-Prot, with manually curated entries; and UniProt/TrEMBL, which is enriched with automated classification and annotation. To search for the UniProtKB Accession numbers for your chosen protein use the EBI Sequence Retrieval System ( SRS ).


InterPro Accession Number       [back2list]
InterPro was created in 1999 as a means of amalgamating the major protein signature databases into one comprehensive resource ( Mulder et al. 2005 ). Today, it is still the most comprehensive repository of information regarding protein families, domains and functional sites and now links to structural databases to provide additional information about specific proteins.


Gene Ontology ID       [back2list]
Gene Ontology describes the properties of protein, the three major classes being Biological Processes, Cellular Component and Molecular Function. A useful resource for Gene Ontology is the website AmiGo which shows the various gene ontologies and annotation for specific species. Gene Ontology is hierarchical because some gene ontology IDs represent very general descriptions of function. On entering a GO ID into the IntAct search, all proteins that share that GO ID will be listed. Such a search strategy might be useful for finding protein with similar functions e.g.. MAP kinase activity ( GO:0004707 ), copper ion transporter activity ( GO:0005375 ).
(Beware, very general terms like “kinase” or “transporter” as these will produce too many hits for IntAct to handle. So chose your description or GO ID carefully)


PubMed ID       [back2list]
PubMed, is a feature of the National Library of Medicine. It includes over 15 million citations for biomedical articles back to the 1950's. Experiments are cross-referenced to the relevant publications. If you want to find in IntAct the protein-protein interaction data from publication simply aquire the PubMed ID and enter this into the IntAct search to access the corresponding experiment/s.


Alternatively...       [back2list]
You can search using descriptions of the proteins functions using the recently developed Advanced Search. Phrases such as “hexokinase”, “phosphorylation”, “mitochondria” will produce a list of proteins which feature these in their descriptions.
