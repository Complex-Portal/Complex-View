What protein(s) interact with proteins with a similar function to my chosen protein?

Protein-protein interaction data from similar or related proteins can provide interesting insights into the protein that you are studying. Investigating what proteins interact with another protein related to the one you are studying can give clues to potential interactors as yet unstudied. To search IntAct for related proteins use Gene Ontology Ids or use the Advanced Search with keywords.

Gene Ontology gives Id to protein properties the three major classes being biological processes, cellular component and molecular function. A useful web resource for gene ontology is the website AmiGo which shows the various gene ontologies in a hierarchical tree structure. To find out if the Gene/Protein that you are interested in has any GO terms associated to it have a look at its UniProtKB entry, enter the Gene/Protein accession into QuickGO or alterntively if you know of any properties that your query protein possess use QuickGO to find the relevant GO Id.

By entering a GO ID into the IntAct search all proteins that share that GO Id will be listed. Such a search strategy might be useful for finding protein with similar functions e.g. MAP kinase activity ( GO:0004707 ), copper ion transporter activity ( GO:0005375 ).

(Beware, very general terms like “kinase” or “transporter” will produce too many hits for IntAct to handle. So chose your description or GO Id carefully)

Alternatively, You can search using descriptions of the proteins functions using the recently developed Advanced Search. Phrases such as “hexokinase”, “phosphorylation”, “mitochondria” will produce a list of proteins which feature these in their descriptions.
