Where can I find statistics about IntAct?

Every month after the IntAct database is released statistics for the data held in IntAct are updated. The data provided are:

  • Number of Proteins
  • Number of Interactions
  • Number of Binary Interactions
  • Number of Experiments
  • Number of Controlled Vocabulary Terms
  • Number of Binary Interactions and Proteins Per Organism
  • Number of Interactions by Indentification Method (experiment)
Go to http://www.ebi.ac.uk/intact/statisticView/do/statistics to see the latest figures.
