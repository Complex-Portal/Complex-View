How can I add data to the IntAct database?

The data in IntAct is curated by experienced biologists specially trained to ensure that the data is as detailed and accurate as possible. Thus, if you wish to add data to IntAct it will need to be be passed onto an IntAct curator.

IntAct encourages researchers to submit data that is to be published. Preferably submissions should be in the PSI MI XML format, however, any format (Word, email) will be accepted. For an example of the PSI MI XML format view one of the files here.

For more details about the PSI-MI format click here.

To discuss submitting data with one of the IntAct team email datasubs@ebi.ac.uk.

 

Please send any questions or suggestions to intact-help@ebi.ac.uk
