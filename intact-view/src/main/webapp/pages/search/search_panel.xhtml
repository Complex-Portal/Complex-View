Search in IntAct         MIQL syntax reference    

Choose a term by clicking on its name. The identifier will be used as the value for the search.


 

Search Tips

  • Free text search will look by default for interactor identifier, (e.g. gene name , UniProtKB Ac or UniProtKB Id ), species, interaction id, detection method, interaction type, publication identifier or author (e.g. Pubmed Id ), interactor xrefs, interaction xrefs.
  • For a more specific search, use MIQL syntax or advanced search
  • Search based on exact word matches e.g. BRCA2 will not match BRCA2B
  • Search for isoforms of 'P12345' by using 'P12345*'
