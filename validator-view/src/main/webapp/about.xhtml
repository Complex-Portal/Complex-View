Local Installation

The PSI-MI validator can be installed locally if the files to validate are too large for the web application. It can also be useful for running batch jobs validating many files in a row.

Write your own validator

The validator framework can be extended/customized to handle semantic validation adapted to your use cases. Please visit our for more information and examples.

MIMIx documentation

The Minimum Information about a Molecular Interaction eXperiment (MIMIx) is a community guideline advising the user on how to fully describe a molecular interaction experiment and which information it is important to capture. More information bout MIMIx can be found
