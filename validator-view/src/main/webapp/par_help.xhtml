2.5.3

This ensures sure that the use of controlled vocabularies is correct. That is, for each controlled vocabulary place-holder, it checks that one uses the appropriate set of terms as defined in the PSI-PAR ontology. The rule is currently looking at:
  • Interaction detection methods
  • Interaction types
  • Participant identification methods
  • Participant's experimental roles
  • Participant's biological roles
  • Participant's Feature detection method
  • Participant's feature type (in PSI-MI or PSI-MOD )
  • Experiment's bibliographical references ( pubmed or DOI )
